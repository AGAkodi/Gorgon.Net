# Gorgon.Net — 0G Bounty Demo Script 🎥

When you record your screen (using QuickTime, Loom, or OBS), follow this 2-minute script to guarantee you hit all the bounty requirements and impress the judges. 

## Preparation
- Make sure the dev server is running (`npm run dev`) and you have `VITE_0G_API_KEY` set in your `.env` file.
- Open `http://localhost:5173` in a fresh browser window.

---

## The Script (2 Minutes)

### 1. (0:00 - 0:30) Introduction & The Omnibox
**Action:** Open the Gorgon.Net home page. 
**Narration:** *"Welcome to Gorgon.Net, an AI-powered Web3 browser secured by the 0G Compute Network. Let's start by doing a risk-assessment search."*
**Action:** Type `"airdrop"` into the search bar and press Enter.
**Narration:** *"Notice the loading spinner — Gorgon is securely querying 0G's minimax-m3 model to generate a live Web3 threat-assessment summary. It instantly warns us about common airdrop phishing vectors."*

### 2. (0:30 - 1:00) Trust Verification & Analytics
**Action:** Click on the **"Uniswap"** search result (or type it into the omnibox). 
**Narration:** *"When we navigate to a verified dApp like Uniswap, the browser runs a real-time safety check and assigns a Green trust score."*
**Action:** Click on the **"0G Net"** tab in the left sidebar.
**Narration:** *"Under the hood, Gorgon is continuously syncing with the decentralized 0G Network. Here you can see the live latency, active edge nodes, and inference logs ensuring our security data remains decentralized and censorship-resistant."*

### 3. (1:00 - 1:30) The Killer Feature — AI Pre-Sign Audit
**Action:** Type `ape-vaults-mint.net` into the URL bar and press Enter. 
**Narration:** *"Now let's simulate a user landing on a phishing site. It claims to offer a free mint."*
**Action:** Click the **"Open In isolated Sandbox"** button on the sidebar. Inside the sandbox, click **"Mint NFT"**.
**Narration:** *"Before the wallet even prompts for a signature, Gorgon intercepts the transaction bytecode. It sends the payload directly to the 0G Compute Network for an instant security audit. Notice how the AI detects a hidden `setApprovalForAll` call, blocking the drainer attempt and protecting the user's assets."*

### 4. (1:30 - 2:00) AI Chatbot
**Action:** Open the **"Ask AI"** tab in the sidebar. 
**Narration:** *"Users also have a persistent Web3 security companion. Powered by 0G's deepseek-v4-flash, it provides ultra-low latency context."*
**Action:** Type: *"What is setApprovalForAll?"* and press Send.
**Narration:** *"The streaming response is blazing fast, providing users with immediate, decentralized security advice without relying on centralized Web2 servers."*
